# The Jabba Initiative

---

## OVERVIEW

---

An educational/entertaining "idle" game about keeping a frog pet on your desktop, right next to your workspace.

> Currently, this is a minimal prototype made to validate an idea with extremely little time (~1.5 hours).

* Code: a couple of Gemini prompts
* Art: by hand
* Idea: with love

## LAUNCH

---

### PREREQUISITES (dev mode):

* NodeJS: v24

> Temporarily no proper Electron build implemented, only developer mode available.

### LAUNCH:

1. Install packages inside root:
```npm install```

2. Run the app using script:
```npm start```
